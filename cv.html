Roblox Anti-Cheat (TAC)

In recent years, I've taken a break from modifying games and focused more on the opposite side : defending against cheats. I've attemped some Unity anticheats but they weren't that interesting as It had to be different for each game, so I've been focusing on Roblox Development and I made this lightweight anticheat which is effective against popular exploits. It's been used in some games and has proven effective. The anticheat is mostly based on server-side checks so circumventing it is extremely difficult.
